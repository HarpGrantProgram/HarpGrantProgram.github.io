HARP Grant Program

Example Timeline

The following is an example of the timeline a single instance of the grant program might follow:

February 22

Application available

March 1

Information session and grant project brainstorming workshop, 9pm Hillside Foothill TV lounge

March 6th

Information session, 6 PM - 8 PM Hillside Foothill TV lounge

March 7th

Early Deadline, Application Due @ 10 PM (you can still apply later by emailing berkeleyharp@gmail.com )

March 12th

Decision Released

March 22th

Project plans finalized and all teams connected with mentors appropriately suited for their project/research.

End of March/Beginning of April

Project Review Panel. Participants will present a finalized plan for the project and provide updates of the project so far

April 30th

Poster session held where recipients who completed the program can show off what they've done. Note that recipients don't have to be entirely finished with their project if it is large, but should have gotten a lot of work done throughout the year. Tentative: Awards and recognitions will be given to outstanding projects.
